# Sales Performance Analytics Dashboard

This project explores revenue drivers, profitability trends, and product performance using interactive Tableau dashboards.

The analysis identifies revenue concentration, evaluates category profitability, and compares sales performance across regions and representatives.

Key features of the project include:

- Executive KPI dashboard summarizing revenue, profit, and sales performance

- Pareto analysis identifying product revenue concentration

- Profitability heatmap comparing performance across regions and product categories

- Revenue vs profit scatter analysis of product performance

- Interactive filters and parameter-driven analysis

Tools Used:
Tableau, data visualization, business intelligence analytics
